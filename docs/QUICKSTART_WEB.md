7. ТАБЛИЦЫ БАЗЫ ДАННЫХ
================================================================================

СПРАВОЧНИКИ (только чтение для пользователя):
  - train      - Поезда (номер, скорость, тип, количество вагонов)
  - platform   - Платформы (номер, вместимость, расположение)
  - service    - Услуги (название, цена, тип)

ОПЕРАЦИОННЫЕ ТАБЛИЦЫ:
  - schedule   - Расписание (рейсы, время прибытия/отправления)
  - ticket     - Билеты (вагон, место, цена, пассажир)
  - passenger  - Пассажиры (ФИО, паспорт, телефон)
  - employee   - Сотрудники (ФИО, должность, стаж)
  - appointment - Назначения на услуги
  - work       - Назначения на поезда

8. ГОТОВЫЕ SQL ЗАПРОСЫ
================================================================================

Расписание:
  SELECT * FROM schedule ORDER BY date, arrival_time

Поезда:
  SELECT * FROM train WHERE speed > 150 ORDER BY speed DESC

Билеты:
  SELECT t.ticket_number, t.seat_number, t.ticket_price,
         p.full_name
  FROM ticket t
  LEFT JOIN passenger p ON t.passenger_number = p.passenger_number

Сотрудники:
  SELECT position, COUNT(*) as count, AVG(work_experience) as avg
  FROM employee
  GROUP BY position
  ORDER BY count DESC

JOIN запрос:
  SELECT s.schedule_number, s.arrival_time, s.departure_time,
         tr.train_number, tr.type, tr.speed
  FROM schedule s
  JOIN train tr ON s.train_number = tr.train_number
  ORDER BY s.date, s.arrival_time

10. УСТРАНЕНИЕ НЕПОЛАДОК
================================================================================

Ошибка "Cannot connect to database":
  - Проверьте, запущен ли PostgreSQL
  - Проверьте параметры подключения в config.json

Ошибка "Superuser authentication required":
  - Войдите как суперпользователь (пароль: 4444)

Веб-клиент не загружается:
  - Проверьте, запущен ли сервер
  - Откройте консоль браузера (F12) для просмотра ошибок
