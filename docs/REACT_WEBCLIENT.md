## 🚀 Установка и запуск

### Требования
- Node.js 16+
- npm или yarn
- Python 3.8+
- PostgreSQL 10+

### 1. Установка зависимостей

```bash
cd C:\VS_Code\RWSDBv2.1\app\web_client-react
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Откройте: **http://localhost:3000**

### 3. Сборка для production

```bash
npm run build
```

Файлы сборки появятся в папке `dist/`.

### 4. Запуск с Python сервером

```bash
# После сборки
cd ../..
python __main__.py
```

Откройте: **http://localhost:8080/web_client/**

## 🧩 Компоненты

### Header
```jsx
<Header
  isSuperuser={isSuperuser}
  onLoginClick={() => setShowLoginModal(true)}
  onLogout={handleLogout}
/>
```

### Navigation
Автоматически показывает вкладку "Бэкап" только для суперпользователя.

### Tables
Полный CRUD интерфейс:
- Просмотр всех таблиц
- Добавление записей
- Редактирование записей
- Удаление записей
- Фильтрация данных

### Queries
- Текстовое поле для SQL запросов
- Готовые запросы для быстрого старта
- Валидация (только SELECT)
- Отображение результатов в таблице

### SpecialQueries
20+ предустановленных запросов по категориям:
- 📈 Статистика и аналитика
- 🚆 Поезда
- 👥 Сотрудники
- 🎫 Билеты и пассажиры
- 🚉 Платформы

## 🔌 API

Все запросы выполняются через `/api`:

### Таблицы
```javascript
// GET - Получить данные
GET /api/train
GET /api/passenger
GET /api/ticket
GET /api/employee
GET /api/platform

// POST - Создать запись
POST /api/train
{
  "speed": 200,
  "year_manufactured": 2024,
  "type": "Скоростной",
  "carriage_count": 10
}

// PUT - Обновить запись
PUT /api/train/1
{
  "speed": 250
}

// DELETE - Удалить запись
DELETE /api/train/1
```

### Запросы
```javascript
// Выполнить SQL запрос
PATCH /api/custom_query
{
  "query": "SELECT * FROM train LIMIT 10"
}
```

### Аутентификация
```javascript
// Вход суперпользователя
PATCH /api/login
{
  "password": "4444"
}

// Ответ:
{
  "success": true,
  "message": "Authentication successful",
  "token": "4444"
}
```

### Бэкап
```javascript
// Создать бэкап (требуется суперпользователь)
POST /api/backup
Authorization: Bearer 4444
```

## 📊 Сравнение версий

| Характеристика | Vanilla JS | React |
|---------------|------------|-------|
| **Архитектура** | Процедурная | Компонентная |
| **Состояние** | Глобальные переменные | useState хуки |
| **Жизненный цикл** | Ручное управление | useEffect |
| **Маршрутизация** | Ручная | Компонентная |
| **Сборка** | Нет | Vite |
| **Оптимизация** | Ручная | Автоматическая |
| **Переиспользование** | Ограничено | Компоненты |
| **Тестирование** | Сложно | Легко |
