# Быстрый старт - React клиент

## Запуск приложения

### Шаг 1: Установка зависимостей

```bash
# Перейдите в директорию клиента
cd client

# Установите зависимости
npm install
```

### Шаг 2: Запуск backend сервера

Откройте терминал и запустите Python сервер:

```bash
# В корневой директории проекта
python server.py
```

Сервер запустится на порту 8080.

### Шаг 3: Запуск React клиента

Откройте второй терминал и запустите клиент:

```bash
# В директории client
cd client
npm run dev
```

Клиент запустится на порту 3000.

### Шаг 4: Откройте браузер

Перейдите по адресу:
```
http://localhost:3000
```

## Сборка для production

```bash
cd client
npm run build
```

Production файлы будут в директории `client/dist/`.

## Решение проблем

### Backend не запускается

Убедитесь, что PostgreSQL запущен и конфигурация в `config.json` верна.

### Клиент не подключается к API

Проверьте, что backend сервер запущен на порту 8080.

### Ошибки при установке

```bash
# Очистите кэш и переустановите
rm -rf node_modules package-lock.json
npm install
```
